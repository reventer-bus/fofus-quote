# FOFUS ₹2,000 Google Ads Safety Plan

**Date:** 2026-07-21  
**Budget:** ₹2,000 maximum (one-time, emergency test)  
**Break-even target:** ₹20,000 in attributable orders  
**Required ROAS:** 10×  
**Risk:** High — if ROAS < 10×, the ₹2,000 is a loss.

---

## Executive Constraint

> "We can have max of ₹2,000 only for the ad. Unless it gets back at least ₹20,000 worth of order, we may get bankrupt."

This means the ad test must be treated like a high-stakes experiment, not a "learning" spend. Every rupee must go to the highest-intent audience with the fastest path to purchase.

---

## Recommended: DON'T spend the ₹2,000 on cold Google Ads yet

**Reason:** ₹2,000 at Google Ads auction prices buys very little traffic, and 10× ROAS on a cold first campaign is unlikely without a proven funnel.

A more reliable path to ₹20,000 with only ₹2,000 is:
- **₹0 on ads** + **manual B2B outreach** using the Google Maps lead scraper.
- OR **₹2,000 on a highly targeted retargeting/remarketing + WhatsApp follow-up** campaign.

If the executive team insists on Google Ads, the safest structure is below.

---

## If Forced to Run Google Ads with ₹2,000

### Campaign choice
**Search-only campaign** (not Performance Max). PMax burns budget across Display/YouTube/Discover where intent is low. Search captures people already looking.

### Budget split
| Allocation | Amount | Purpose |
|---|---|---|
| Search ads | ₹1,500 | High-intent keywords |
| Call extensions / ad assets | ₹0 | Use free extensions |
| UTM tracking / landing page | ₹0 | Already built |
| Contingency / day-2 refill if winning | ₹500 | Only if first 2 days show clicks + quote requests |

### Keywords (single keyword ad groups)
Target only exact-match and phrase-match high-intent terms:

- `"3d printing thrissur"`
- `"3d printing service thrissur"`
- `"custom 3d printing kerala"`
- `"3d printed gifts thrissur"`
- `"3d printing near me"`
- `"architectural scale model thrissur"`
- `"candle mold 3d printing"`
- `"3d printed idol kerala"`

**Negative keywords:** free, cheap, STL download, thingiverse, job, salary, course, printer price, buy printer, DIY.

### Landing page
`https://quote.business.fofus.in` — direct to quote form so intent converts fast.

### Tracking
- UTM: `?utm_source=google&utm_medium=search&utm_campaign=fofus-2k-test`
- Each quote request must ask "How did you hear about us?" → confirm Google Ads.
- Shopify orders tagged with the same UTM.

### Bidding
- Manual CPC, max ₹15/click.
- ~100 clicks possible with ₹1,500.
- Need 1 in 10 clicks to request a quote, and 1 in 5 quote requests to place a ₹1,000+ order to hit ₹20,000.

### Schedule
- Run only **10am–8pm**, all days.
- Pause immediately if no quote requests after ₹500 spend.

### Daily cap
- ₹500/day max for 4 days.
- Kill switch: if after ₹1,000 spent there are zero paid orders, pause and report failure.

## Alternative Zero-Risk Plan (Recommended)

Use the ₹2,000 for **operational leverage**, not ad spend:

| Use | Amount | Expected return |
|---|---|---|
| Print 50 high-quality FOFUS sample cards / mini-catalogues | ₹1,000 | Distribute to architects, gift shops, schools |
| ₹500 worth of WhatsApp Business API / automation tool credits | ₹500 | Faster follow-up of warm leads |
| ₹500 reserve for urgent shipping/quote incentives | ₹500 | Close one hot lead |

This avoids the auction risk entirely and converts existing local relationships.
